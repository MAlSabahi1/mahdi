# Admin Dashboard

A simple, clean admin dashboard built with Vue.js 3 and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js 18.x or later (recommended to use Node.js 20.x or later)

### Installation

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Technologies Used

- **Vue.js 3** - Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **TypeScript** - Type Safety
- **Vue Router** - Routing
