# الدليل الهيكلي البصري الدقيق للكشوفات والتقارير الورقية (مطابق لصور الدليل الإرشادي)

يوضح هذا الدليل الهيكل والتصميم الهندسي الدقيق لجداول التقارير والصفحات الورقية الرسمية كما هي مطبوعة في الدليل الإرشادي الفعلي لضبط الخدمات، بناءً على المطابقة البصرية المباشرة للصور المرفقة.

## ⏳ ثانياً: كشوفات القوة غير العاملة مؤقتاً (الفترات والتفاصيل)

### 4. نموذج رقم (6): كشف بالقوة غير العاملة مؤقتاً مرافقين
* **المطابقة البصرية**: الجدول ينتهي عند عمود (مدة التفريغ) ولا يحتوي على عمود ملاحظات.
* **رأس الجدول المطبوع**:

| م | الرتبة | الرقم العسكري | الاسم | مصدر الأمر | اسم الشخصية | منصب الشخصية | مدة التفريغ (عمود مدمج) | |
|---|---|---|---|---|---|---|---|---|
| | | | | | | | **من** | **إلى** |

* **التوضيحات والضوابط المطبوعة أسفل الصفحة**:
  * **توضيح**: عمر الوزير فقط هي مرافقين مدير شرطة المحافظة (إذا كان مرافق داخل شرطة المحافظة) القوة عاملة.
  * **اسم الشخصية**: اسم طالب التفريغ للمرافقة بمعيته.
  * **مصدر الأمر**: وزير الداخلية، نائب الوزير، الوكيل لقطاع الموارد، (المدير العام).
  * **منصب الشخصية**: العمل الذي يقوم به طالب التفريغ.
  * **مدة التفريغ**: تحديد بداية التفريغ ونهايته.
  * **حاشية الاعتماد**: يعتمد كشف حراسة ومرافقة مدير شرطة المحافظة والفروع والمصادقة عليه.
